Desarrollo de un pipeline completo de detección de fraude en transacciones de Bitcoin utilizando el dataset Elliptic. Se realizó un análisis exploratorio exhaustivo de la red temporal de transacciones (49 snapshots, 165 features por nodo), evaluando distribuciones de clases, conectividad, evolución temporal y reducción de dimensionalidad (PCA, UMAP). Posteriormente, se aplicó una curación rigurosa de datos (eliminación de nodos aislados y etiquetas desconocidas) y se compararon 8 modelos de clasificación, incluyendo métodos tradicionales (Regresión Logística, Random Forest, SVM, Árboles, XGBoost), redes neuronales (MLP) y modelos basados en grafos (GCN, GAT). Se validaron los modelos con K-Fold estratificado, obteniendo que XGBoost alcanzó el mejor rendimiento (F1-score ~0.94), superando a los modelos de grafos debido  que los atributos individuales resultaron más informativos que la estructura topológica para este conjunto de datos.
